# Monster8
Monster8 is a Fantasy Console.

There are two programs:
- The fantasy console itself (emulator)
- An assembler editor

- GTK 3.0
- GtkSourceView 3.0 (optional)
- HarfBuzz
- C++17 compatible compiler (GCC 7+)
- Make
- pkg-config
